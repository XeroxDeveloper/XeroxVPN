## Киллер-фичи (MetaFix Update)

### Интеллектуальный Split Tunneling
Уникальная функция, вдохновленная современными трендами: вы сами выбираете, на какие процессы будет влиять защита. Настройте VPN только для **выбранных игр или приложений** (до 10 одновременно), сохраняя максимальную скорость и минимальный пинг для всей остальной системы.

### Динамический интерфейс «Speedometer»
Забудьте о скучных кнопках. В Xerox VPN центр управления трансформируется в живой анимированный спидометр, который в реальном времени отображает пропускную способность канала и состояние вашего туннеля.

### Продвинутая маскировка и Stealth Mode
*   **Stealth Fragment:** Умное дробление пакетов данных, делающее ваш трафик невидимым для систем анализа провайдера (DPI).
*   **Multi-Hop Chain:** Возможность построения зашифрованных цепочек через несколько стран для абсолютной анонимности.
*   **WebRTC Shield:** Глубокая системная блокировка утечек вашего реального IP.

## Техническое превосходство

- **Engine:** Нативное ядро на Kotlin с использованием системных ресурсов Android VpnService.
- **UI System:** Собственная разработка в стиле Material You 3 с поддержкой динамических пузырьков и реакцией на гироскоп устройства.
- **Full Architecture Support:** Идеальная работа на любых устройствах благодаря поддержке `armeabi-v7a` и `arm64-v8a`.
- **Zero-Log Policy:** Мы не собираем и не храним ваши данные. Всё, что происходит в туннеле, остается внутри туннеля.

## Структура проекта

*   `XeroxVpnService.kt` — Высокопроизводительное ядро туннелирования.
*   `Servers.kt` — Глобальная база оптимизированных серверов (Япония, Германия, Россия и др.).
*   `MainActivity.kt` — Центр управления и интеграции системных вызовов.
